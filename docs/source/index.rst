coretp
===================================================

RISC-V Core Test Plan (``coretp``) is a tool for describing test plans in both human readable format and machine parsable format.
The tool is designed to read in some asciidoc formatted test plan, and represent the test plan in a Python object.
The Python object can then be consumed by other tools such as a test generators, coverage tools, and documentation.
